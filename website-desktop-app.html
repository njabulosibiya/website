Design

application architecture

Since the website is already live, I could just inspect its dimensions from chrome developer. However, I shall redesign it from scratch. The preffered UI design tool would be figma as it is free and offers great flexibility and easy when designing. The first stpe of the development of this application, of any application would be design. Not just for user interface but OOP design patterns as well.

The application design process shall be divided into 2 processes that can be done concurrently. These are the UI as mentioned & the application logic. As C++ is currently the only language I am well versed in, I shall utilize it.

NOTE:

QML User Interface shall use primitize design patterns as the knowledge of QML is intermediate at best. Once higher level QML has been learnt, application shall be updated.
